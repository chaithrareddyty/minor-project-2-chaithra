# minor-project-2-chaithra
The Solution: SpendDNA
​This project is a 7-day coding pipeline built in Python to automatically clean up that financial mess and tell a story about the user's spending habits.  
​Here is how it works step-by-step:
​Day 1: Clean Up – It deletes duplicate rows, fixes broken dates, strips away messy currency symbols, and separates spending (debits) from income (credits).  
​Day 2: Identify Vendors – It looks at transaction descriptions and maps them to clean, recognizable names (e.g., turning AMZN into Amazon). It also separates personal UPI transfers from commercial store purchases.  
​Day 3 & 4: Categorize & Track – It groups your spending into 12 main areas (like Food Delivery or E-commerce) and calculates your monthly trends and savings rate. It even tracks when you spend money, like flagging late-night food orders.  
​Day 5: Find Unusual Spending – It uses math (Z-scores) to catch "anomalies"—unusually large purchases that don't match your normal habits.  
​Day 6 & 7: Give a Personality Report – Finally, it labels the user with a "Spending Archetype" based on their habits, such as The Foodie, The Shopaholic, or The Investor.  
​📊 The Final Output
​The system generates an easy-to-read text dashboard. For example, when running the data for a user named Rahul, it revealed that while he saves a healthy 15.58% and invests well, he also had a massive, unusual 18,000 rupee uncategorized expense
